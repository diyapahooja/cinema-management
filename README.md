# 🎬 Cinema Management System

A fully functional desktop application built with **JavaFX** for managing a cinema's movie catalog, user roles, and ticket bookings. This project demonstrates object-oriented principles, GUI development, and file handling.

## ✨ Features

### 👑 Admin Panel
- **Add Movie** – Add new movies with title, genre, and description.
- **Delete Movie** – Remove existing movies from the catalog.
- **View Movies** – Display the complete movie list.
- **Manage Ticket Pricing** – Update ticket prices dynamically.
- **Manage Movie Categories** – Filter and browse movies by genre.

### 🧑‍🎤 Customer Panel
- **View Movies** – Browse all available movies.
- **Add to Favorites** – Select and save favorite movies.
- **View Favorites** – See your saved favorites list.
- **Book a Ticket** – Choose a movie and seat number to book.
- **View Receipt** – See your last booking details with price.
- **Download Receipt** – Save your booking receipt as a `.txt` file.
- **Show by Genre** – Filter movies by specific genres.

### 🔐 Authentication
- Secure login system with two default users:
  - **Admin**: `admin` / `admin123`
  - **Customer**: `diya` / `diya`
